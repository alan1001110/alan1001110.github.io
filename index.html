My name is Alan and I'm learning to code. I'd like to share my journey.

As my very first step down the programming path, I delved into Learn Python The Hard Way by Zed Shaw. A friend had told me Python would be a good language to learn first, and Google had told me a lot of people thought LPTHW would be a good book for learning Python.

When I started getting bored with the book, I googled, "What programming language should I learn first?" What I found always boiled down to either, "It doesn't matter" or "Learn Javascript first." So I started learning Javascript instead, and promptly forgot what Python I'd learned thus far.

My primary resource for Javascript learning has been Free Code Camp. I'm loving it. It lays out a complete track for learning web development from start to finish, and ends with doing coding projects for non-profits. You can take a look my Free Code Camp activity here.
